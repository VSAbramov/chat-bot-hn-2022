Быстрое выполнение определенных сложных задач простым способом обычно является важным
принципом для создания языков сценариев. Исходя из этого принципа, языки сценариев обычно
проще и проще, чем языки системного программирования, такие как C, C ++ или Java.
Некоторые особенности Python - это языки сценариев:
• Синтаксис и структура обычно относительно просты;
• Обычно его легко изучить и использовать;
• «Интерпретация» программы, которую легко изменить, обычно используется в качестве
рабочего режима без необходимости «компиляции»;
• Возможности разработки программы лучше, чем рабочие характеристики.
Сценарий может автоматизировать интерактивные операции, которые в противном случае
выполнялись бы с помощью клавиатуры. Сценарий оболочки в основном состоит из команд,
которые изначально необходимо вводить в командной строке или в текстовом редакторе,
пользователи могут использовать сценарии для объединения некоторых общих операций в набор
серийных номеров. Язык, который в основном используется для написания этого сценария,
называется языком сценария. Многие языки сценариев фактически превосходят простые
последовательные инструкции пользовательских команд, а также могут писать более сложные
программы.
IDLE - это оболочка Python. По сути, это способ взаимодействия с программами путем ввода
текста! Подобно окну cmd в нашей Windows, как и темное командное окно Linux, все они являются
оболочками. Используя их, мы можем отдавать команды операционной системе. Точно так же мы
можем использовать оболочку IDLE для взаимодействия с Python.
BIF - это встроенные функции. Чтобы помочь программистам быстро писать сценарии, Python
предоставляет множество встроенных функций, нам нужно только вызывать их напрямую,
например, функция print () предназначена для «печати на экран», а функция input () - для
получения пользовательского ввода .
